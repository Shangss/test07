# Leaflet Moving Marker

### Usage

##### Installation
```
npm install --save leaflet-moving-marker
```

##### API
```ts
Leaflet.movingMarker(latLng: L.LatLng, {
    endLatLng?: L.LatLng;
    duration?: number;
})
```

### Development

`npm install` and `npm start` to watch for changes and see the results in browser.
